Welcome to the EXTM3U Playlist Creator!

Easily create and manage your M3U playlists with these features:

  • Add **individual channels** with names, URLs, and custom logos.
  • **Automate from Pixeldrain Folder Links:** Simply paste a Pixeldrain folder link (e.g., `https://pixeldrain.com/l/FOLDER_ID`), and all files within that folder will be added as individual channels with their API links!
  • **Smart Logo Fetching:** If you add a channel containing "movie" in its name and no custom logo, we'll try to fetch a relevant movie poster from TMDB!
  • Set **global default logo and group names** for all channels. These will be saved automatically!
  • **Search and manage** your added channels with ease.
  • **Bulk Add Channels:** Add multiple channels quickly by pasting them in a specific format.
  • **Import M3U:** Upload an existing `.m3u` file to populate your channel list.
  • **Save and load multiple playlists** locally in your browser.
  • **Generate and download** your M3U playlist file instantly.
  • **Copy** the generated M3U output to your clipboard.

Get started by adding your first channel or Pixeldrain folder link!
